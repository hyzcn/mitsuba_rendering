### Rendering 
1. Install [Mitsuba](https://www.mitsuba-renderer.org/).
2. Activate Mitsuba: `source where-installed-mitsuba/setpath.sh`
2. Render round table video: `python main.py --meshfolder ./mesh --roundtable`
